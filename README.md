# cmm-xhs-auto-MiniProgram

# common
通用模块，放置常用的方法
# conf
配置模块，放置环境配置，页面路径类文件
# outputs
输出模块，放置运行日志，截图，数据等
# page
页面封装模块，basepage放置基础页面操作等方法，xxxpage放置页面元素，操作方法，业务流程
# testcase
测试用例模块，basecase放置基础用例的前置后置等，test_xxx放置测试用例

vip_6是企业版用例，vip_5是个人版用例，vip_4是专业版用例，vip_1是普通用户的用例
# congig文件
xxx_config.json文件为运行ide，真机所需要配置的参数
# main文件
运行文件
# suite.json文件
测试套件，放置测试规则
